102.二叉树的层序遍历

  • 给你一个二叉树，请你返回其按 层序遍历 得到的节点值。 （即逐层地，从左到右访问所有节点）。
示例：
二叉树：[3,9,20,null,null,15,7],
    3
   / \
  9  20
    /  \
   15   7
返回其层次遍历结果：

[
  [3],
  [9,20],
  [15,7]
]

深度度优先遍历-DFS

class Solution {
    List<List<Integer>> depthList = new ArrayList<>();

    /**
     * 解法一: 采用深度优先遍历
     * @param root 根节点
     * @return List<List<Integer>>
     */
    public List<List<Integer>> levelOrderUseDFS(TreeNode root) {
        if (null == root) {
            return depthList;
        }
        recursive(root, 0);
        return depthList;
    }

    public void recursive(TreeNode node, int depth) {
        if (depthList.size() == depth) {
            //这里使用链表，不需要扩容
            depthList.add(new LinkedList<>());
        }
        depthList.get(depth).add(node.val);
        if (null != node.left) {
            recursive(node.left, depth + 1);
        }
        if (null != node.right) {
            recursive(node.right, depth + 1);
        }
    }

}

广度优先遍历-BFS

     /**
     * 解法二: 采用深度优先遍历
     * @param root 根节点
     * @return List<List<Integer>>
     */
    public List<List<Integer>> levelOrderUseBFS(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (null == root) {
            return res;
        }
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        while (!queue.isEmpty()) {
            int size = queue.size();
            List<Integer> list = new LinkedList<>();
            //内循环取出位于二叉树同一层的元素
            for (int i = 0; i < size; i++) {
                TreeNode cur = queue.remove();
                list.add(cur.val);
                if (null != cur.left) {
                    queue.add(cur.left);
                }
                if (null != cur.right) {
                    queue.add(cur.right);
                }
            }
            res.add(list);
        }
        return res;
    }
